# Creative Line Graphics Website Redesign & Content Update

## Project Overview

Redesign and modernize the existing printing shop website:

Website:
https://creative-line-graphics.vercel.app/

Business Name:
Creative Line Graphics

This is a professional printing shop website.  
The new design should look modern, premium, clean, responsive, and professional.

The goal is:

- Improve UI/UX
- Modernize branding
- Add dark/light mode toggle
- Improve content structure
- Add better typography and color system
- Improve mobile responsiveness
- Make services visually attractive
- Improve CTA sections
- Add smooth animations
- Improve SEO structure

---

# Tech Stack

Use:

- Next.js

- Framer Motion (for animations)
- Lucide React Icons
